# üåç üìû Country Dial Codes API  

Welcome to the **Country Dial Codes API**, powered by **JSONly**.  
This API provides accurate and up-to-date information about **international dialing codes**, including each country's **name, ISO code, and calling prefix** (for example, `+91` for India) ‚Äî all in a fast, lightweight JSON format.  

Use this API to seamlessly integrate **global calling code data** into your applications, authentication systems, or contact management tools.
 
 ---

## üîó Base URL

```bash
GET https://jsonly.vercel.app/api/countries/dial"
```
> **üü¶ Note:**  
> By default, the API returns **10** results per request.  
> You can modify this using the **`limit`** parameter as needed.  
>  
> Country names containing spaces must be URL-encoded using **`%20`**.  
> Example: `united states ‚Üí united%20states`  

This is the **root endpoint** for all **country dial code‚Äìrelated API calls**.

## üß© Query Parameters

| Parameter  | Type     | Required | Description                                                                                         |
| ----------- | -------- | -------- | --------------------------------------------------------------------------------------------------- |
| `country`   | `string` | üü¢ Yes    | Name, ISO country code, or dial code to search for (case-insensitive). Example: `"India"`, `"US"`, `"+91"`. |
| `limit`     | `number` | üî¥ No     | Defines how many country dial codes are returned per request. Useful for managing pagination or response size. |
| `offset`    | `number` | üî¥ No     | Skips a specified number of results before returning data. Commonly used with `limit` for pagination. |
