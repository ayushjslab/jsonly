# üåç üè≥Ô∏è Country Flags API

Welcome to the **Country Flags API**, powered by **JSONly**.  
This API provides detailed information about **country flags**, including each country's **name, ISO code, and flag image URL** ‚Äî all served in a fast and structured JSON format.

Use this API to easily integrate country flag data into your applications, dashboards, or internationalization tools.


## üîó Base URL

```bash
GET https://jsonly.vercel.app/api/countries/flag"
```
> **üü¶ Note:**  
> By default, the API returns **10** results per request.  
> You can modify this using the **`limit`** parameter as needed.  
>  
> Country names containing spaces must be URL-encoded using **`%20`**.  
> Example: `united states ‚Üí united%20states`  

This is the **root endpoint** for all **country dial code‚Äìrelated API calls**.

## üß© Query Parameters

| Parameter  | Type     | Required | Description                                                                                         |
| ----------- | -------- | -------- | --------------------------------------------------------------------------------------------------- |
| `country`   | `string` | üü¢ Yes    | Name or ISO country code to search for (case-insensitive). Example: `"India"`, `"US"`, `"Japan"`.   |
| `limit`     | `number` | üî¥ No     | Limits how many country flags are returned per request. Useful for controlling response size.       |
| `offset`    | `number` | üî¥ No     | Skips a specified number of results before returning data. Typically used with `limit` for pagination. |
