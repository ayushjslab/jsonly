# üáÆüá≥ üöÜ Indian Railway Stations API

Welcome to the **Indian Railway Stations API**, powered by **JSONly**.  
This API allows you to fetch detailed data about railway stations across India ‚Äî including **station codes, names, and associated cities** ‚Äî all in a fast and structured JSON format.

Use this API to integrate Indian railway station data into your applications, dashboards, or travel tools with ease.


---

## üîó Base URL

```bash
GET https://jsonly.vercel.app/api/india/railway-stations"
```
> **Note:**  
> By default, the API returns **10** results per request.  
> You can adjust this using the `limit` parameter as needed.  
>  
> Spaces in URLs are encoded as `%20`.  
> Example: `new delhi ‚Üí new%20delhi`


This is the root endpoint for all railway staion realted API calls.

## üß© Query Parameters
| Parameter | Type     | Required | Description                                                                                           |
| ---------- | -------- | -------- | ----------------------------------------------------------------------------------------------------- |
| `station`  | `string` | üü¢ Yes    | Search query for the station name, station code, or city (case-insensitive). Example: `"Delhi"`, `"NDLS"`, `"Mumbai"` |
| `limit`    | `number` | üî¥ No     | Maximum number of stations to return per request. Useful for pagination or limiting results.          |
| `offset`   | `number` | üî¥ No     | Number of stations to skip before returning results. Combine with `limit` for pagination.            |
